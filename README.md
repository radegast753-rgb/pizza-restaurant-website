# Pizza Restaurant Website with Admin Panel

The project consists of two interconnected pages: a customer interface and a control panel.

## Live Demo
* **Main Website (Menu):** https://radegast753-rgb.github.io/pizza-restaurant-website/
* **Admin Panel:** https://radegast753-rgb.github.io/pizza-restaurant-website/admin.html

## How It Works
1. **Main Page (index.html)** – A dark-themed restaurant landing page featuring the menu, pricing, and contact information.
2. **Admin Panel (admin.html)** – A page where you can modify menu items. The changes are immediately reflected on the main website.

Since this project is hosted on GitHub Pages without a backend database, all updates made in the Admin Panel happen directly in the browser via JavaScript and will reset upon page reload. This serves as a functional UI/UX prototype.

## Key Features
* **Page Interconnectivity:** The admin panel and the main menu dynamically interact with each other using JavaScript.
* **Fully Responsive:** The layout renders properly across both desktop monitors and mobile screens.
* **Tech Stack:** Clean HTML5, CSS3 (Flexbox, Grid), and vanilla JavaScript.
